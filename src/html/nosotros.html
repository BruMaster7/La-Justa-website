¿Qué es "La Justa"?

"La Justa" es más que un portal de noticias, es una ventana hacia la información presentada de manera clara, directa y sincera. Su nombre juega con el modismo rioplatense "te canto la justa", una expresión que refleja honestidad y franqueza al compartir la verdad sin adornos. Aquí, cada noticia busca mantenerte informado con integridad, porque en "La Justa", nuestra misión es cantarte las cosas tal como son.

¿De dónde salen nuestras noticias?

Esta plataforma combina el poder del scraping con un diseño limpio y eficiente. Mostramos noticias scrapeadas en tiempo real desde Montevideo Portal, El País y Xataka, demostrando cómo la automatización puede mejorar la experiencia de acceso a información.

La API detrás del proyecto

La API NotiScraperAPI, alojada en GitHub, combina Python, FastAPI, MongoDB y Rye para el manejo de dependencias. Una solución robusta para scraping, almacenamiento y entrega de datos.
